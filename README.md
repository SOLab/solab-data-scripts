solab-data-scripts
==================

Python and Matlab script templates to read data from SOLab servers



point_data.py
-----------------
Export data points to xlsx (Excel). Supported 'wind_speed', 'sst', 'atmosphere_water_vapor_content', 'icecon' parameters.

**requirements:**
* pyxml
* urllib
* xlsxwriter
* pydap
* scipy
* numpy
